七、老师使用时的注意事项
	1.如果因为仪器故障或其他意外情况导致学生因“迟到”、“求助”、“未通过”而扣分，教师需要在评定分数时基于调整
		· 操作分数从满分一百开始减
		· 迟到-5，迟到可以界定为从上课时间开始到上课开始十五分钟内。
		· 未完成：如果一直到下课，学生仍没有提交数据，实验算为未完成，成绩为0
		· 求助次数和未通过次数，每次累计2分。
	2.database.calss.php 函数pass()只适用不超过2个实验的情况
